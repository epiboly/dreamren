如果想成功，必须重视时间的价值！

时间管理的科学方法：

一、 列出代办事项的清单
1. 每周计划或每月计划列明的工作
2. 前一天未完成的工作
3. 日常工作
4. 约会

将时间分为四部分进行规划：
1. 输入时间——自我提升：读书、学习的时间；
2. 输出时间——工作时间：工作、会议的时间；
3. 生活时间——必要时间：吃饭、睡觉的时间；
4. 私人时间——休息时间：约会、休闲的时间；

二、 决定优先顺序、删除和授权（要在时间的要求和事情重要程度两个方面来考核）
1. 非常重要的事
2. 紧急重要——必须解决
3. 紧急不重要——可否授权或删除
4. 不紧急重要——近期解决
5. 不紧急不重要——可授权或删除
6. 合理限制数量——每个人每天的时间的精力有限

三、 估计工作需要的时间
1. 限定自己必须在规定时间内完成工作
2. 全心投入，排除干扰

四、 预留时间给突发事件
安排工作的时间不要超过75%，要留出至少25%的机动时间来处理突发事件。如果你的工作时间是8小时，那么就要留出至少2小时来处理突发事件。

五、 进行每日检查，列出明日事项
不要考验自己的记性，做好明日的计划表后，你可以睡得安稳一些；否则的话，可能整晚你的脑子里都想着：“别忘了！别忘了！别忘了！”
